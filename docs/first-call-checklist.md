# First Call Checklist

Use this checklist before asking a teammate to debug an integration.

- Account created at https://originstartai.com?utm_source=github&utm_medium=docs&utm_campaign=first_call_checklist
- API key generated.
- `ORIGINSTARTAI_API_KEY` is set.
- `ORIGINSTARTAI_BASE_URL` ends with `/v1`.
- `ORIGINSTARTAI_MODEL` is enabled for the account.
- One curl request works.
- One Python or Node.js request works.
- Credits are available in the console.

New users can recharge `$5` and get `$5` extra API credits.
